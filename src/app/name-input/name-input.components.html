What Is League Winrate Estimator?

One of the most frustrating things in League Of Legends is getting a bad ranked team with little chance of winning. Although many websites and apps out there will quickly and easily give you all the stats you would ever want on your team, it is difficult to know how these different data points interact and if it's going to be a close game or a complete stomp. League Winrate Estimator helps you make informed decisions by considering all these factors and giving you a percentage change of winning a game before champ select ends.
League Winrate Estimator is a machine learning-based web application that takes in the names and champions of players in a ranked lobby and predicts which team will win. The win rate for a given lobby is calculated using each player's ranked win rate, champion overall win rate, and player-specific champion win rate. The currently deployed version of the model was trained without any enemy player information so that it can be used in champ select.
This "One Sided" model was trained using ~3600 ranked games (~400 games from each division ranging from Gold 1 to Diamond 1), and preliminary testing of 300 randomly selected games from Iron 4 to Diamond 1 gave an 82% accuracy rate in predicting the correct outcome.

More Info

League Winrate Estimator is not a crystal ball and can only give you an idea of how difficult a game will be. League of Legends is a complex game, and the outcome cannot be solely predicted from champ select. It is still up to you as the player to decide whether to dodge or not.
